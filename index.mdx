## Solicitando acesso

Solicitação de Acesso
Para começar a utilizar a API da Next Shipping, siga os passos abaixo:

**1. Entre em contato com o seu vendedor de conta**\
 Informe que você deseja acesso à API.

**2. Aguarde o retorno do time de vendas**\
 Nosso representante entrará em contato para fornecer seu Client ID e Client Token.

**3. Configure os headers de autenticação**\
`x-client-id`: seu Client ID\
`x-client-token`: seu Client Token

Com essas credenciais em mãos, você já pode consumir todos os endpoints disponíveis.
